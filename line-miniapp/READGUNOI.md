## WorkFlow

#ทำงานในโฟลเดอร์ src เป็นหลัก

-asset -> ใช้เก็บ รูปหรือ asset ที่ใช้ในงาน
-component -> ใช้เก็บ component ของหน้าเว็ยบ เช่น Footer, Header หรืออะไรที่มันซ้ำกันแล้วเอามา re-use กับหน้าอื่นได้
-libs -> ใช้จัดการกับทางฝั่ง Line SDK
-pages -> ใช้จัดการ Flow ของแต่ละเพจ
-utils -> ใช้เชียนฟังก์ชั่นช่วย
